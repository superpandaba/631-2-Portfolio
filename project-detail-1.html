个人博客网站

该项目是一个从零到一构建的现代化个人博客平台。它不仅拥有一个清爽、易于阅读的界面，还实现了一个博客平台的核心功能：文章的发布、展示与管理。 技术上，项目前端运用了HTML5、CSS3和原生JavaScript，实现了页面的动态渲染和响应式布局；后端则通过模拟API的方式，处理文章数据的增删改查，完整地演示了一个全栈应用的数据交互流程。

使用的技术:

  • HTML5, CSS3, JavaScript
  • Node.js (用于模拟API)
  • 响应式设计
  • RESTful API 交互
